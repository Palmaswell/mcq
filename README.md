# mcq

Meta CSS Queries

## Requirements

🚀 Node.js >= 10
🌲 Git
🐈 yarn >= 1.12

## Getting started

```sh
  yarn add mcq
```

## Development

```sh
git clone git@github.com:Palmaswell/mcq.git

# Install packges
yarn
# Start development mode
yarn dev
# Run and watch Jest unit tests
yarn test --watch
```
